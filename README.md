# pytntp
Tiny library to read .tntp (Transportation Network Test Problem) files and convert into NetworkX graphs.

## Installation
```
pip install git+https://github.com/ben-hudson/pytntp
```

## Example usage
See [example.py](example.py) for a complete end-to-end example that loads a TNTP network, computes link congestion, and renders a two-panel visualization.
