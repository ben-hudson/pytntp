# API reference

Reference for every public function in `tntp`, generated from the source
docstrings. Pick a module:

- [`tntp.read`](read.md) — read network, node, flow, and demand files.
- [`tntp.convert`](convert.md) — convert node/edge tables to a NetworkX graph.
- [`tntp.transform`](transform.md) — split zone nodes into source/sink pairs.
- [`tntp.vis`](vis.md) — visualization helpers.
